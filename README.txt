Εντολή μεταγλώττισης:make
Εντολή για διαγραφή παραχθέντων αρχείων από το makefile:make clean

Εντολή εκτέλεσης:./ecosystemSimulator
Αφού το εκτελέσετε, θα σας ζητηθεί να δώσετε τα δεδομένα για να ξεκινήσει η προσομοίωση(μέγεθος grid,αρχική εποχή,πλήθος ημερών για την προσομοίωση, πλήθος για κάθε είδος ζώου και φυτού κλπ).
Επιπλέον, υπάρχει και η δυνατότητα αν θέλετε να επιλέξετε όταν γίνεται η εκτύπωση του οικοσυστήματος να εμφανίζονται αναλυτικές πληροφορίες για τις τιμές όλων των χαρακτηριστικών κάθε φυτού και ζώου που υπάρχει
και είναι ζωντανό εκείνη τη στιγμή στο οικοσύστημα.Προσοχή όμως γιατί αυτό κυρίως για μεγάλα terrain και μεγάλο πλήθος ζώων και φυτών ενδέχεται να εκτυπώσει υπερβολικά μεγάλο όγκο πληροφορίας στην έξοδο με αποτέλεσμα
να μην είναι δυνατή η επιθεώρηση της προσομοίωσης από την αρχή.Για αυτή την επιλογή, γίνεται σχετική ερώτηση στο πρόγραμμα όταν γίνεται η εισαγωγή των δεδομένων.Ακόμα, υπάρχει η δυνατότητα να δώσετε και το δικό σας
seed στο πρόγραμμα σε σχετική ερώτηση στην είσοδο δεδομένων(δίνετε 0 αν θέλετε το seed να είναι τυχαίο).
Μετά την ανάγνωση των δεδομένων, θα τρέξει η προσομοίωση του οικοσυστήματος για το πλήθος ημερών που δώσατε.
Σημείωση:Για κάθε δεδομένο που δίνετε,η ανάγνωση θα επαναλαμβάνεται μέχρι να δοθεί έγκυρη τιμή(έλεγχος εγκυρότητας).Επίσης σε περίπτωση που δεν υπάρχει διαθέσιμος χώρος για κάποιο συγκεκριμένο είδος φυτού, η
προσομοίωση δεν θα εκτελείται και θα εμφανίζεται το αντίστοιχο μήνυμα προειδοποίησης.

Όσον αφορά τη μεθοδολογία που ακολούθησα, αρχικά αφού τα φυτά και τα ζώα έχουν κάποια κοινά χαρακτηριστικά(όνομα,θέση,token κλπ)έφτιαξα μια κλάση Entity για κάθε ζωντανό ον την οποία κληρονομούν οι κλάσεις για
τα φυτά(Plant) και τα ζώα(Animal).Για τα φυτά έφτιαξα 2 διαφορετικές κλάσεις, μία για κάθε ξεχωριστή κατηγορία, μια για αυτά που έχουν καρπούς(Seeded) και μια για αυτά που δεν έχουν(Seedless),οι οποίες κληρονομούν
τη κλάση φυτό(Plant).Έπειτα για κάθε είδος φυτού, έφτιαξα μία διαφορετική κλάση,όπου η κάθε μία από αυτές κληρονομεί την κλάση της κατηγορίας που αντιστοιχεί(π.χ. το γρασίδι κληρονομέι τη Seedless και το πεύκο τη Seeded).
Την ίδια λογική ακολούθησα και για τη δημιουργία των κλάσεων για τα ζώα, δηλαδή έκανα μία κλάση για τα φυτοφάγα(Herbivore) και μία για τα σαρκοφάγα(Carnivore)από τις οποίες και οι δύο κληρονομούν την κλάση ζώο(Animal).
Έπειτα για κάθε είδος ζώου, έφτιαξα μία διαφορετική κλάση, όπου η κάθε μία από αυτές κληρονομεί την κλάση της κατηγορίας που αντιστοιχεί(π.χ. το ελάφι κληρονομεί την Herbivore και ο λύκος την Carnivore).Επιπλέον,όσον
αφορά τον MapGenerator,σύμφωνα με την εκφώνηση κατάλαβα ότι είναι ξεχωριστή οντότητα,γι αυτό το μοντελοποίησα σαν μία ξεχωριστή κλάση η οποία όμως είναι friend class της κλάσης του οικοσυστήματος προκειμένου να έχω
άμεση πρόσβαση στα data members αυτής και επειδή φυσικά σχετίζονται σε πολύ μεγάλο βαθμό μεταξύ τους. Γενικά,όλες οι μέθοδοι όλων των κλάσεων κάνουν ότι ακριβώς περιγράφεται στην εκφώνηση, με εξαίρεση τις επιπλέον
μεθόδους που πρόσθεσα εγώ.Όπου δεν υπάρχει σχόλιο, είναι φανερή η λειτουργία και η χρησιμότητα, και γενικά σε μεθόδους που υλοποίησα εγώ, και όπου έχω κάνει παραδοχές, έχω βάλει σχόλια.

Οι κύριες παραδοχές που έκανα είναι οι εξής:
-Τα ζώα που βρίσκονται σε χειμερία νάρκη, δεν τρώνε, δεν αναπαράγονται, δεν κινούνται, δεν μπορούν να φαγωθούν από άλλα ζώα, δεν επαναφέρεται η πείνα τους και δεν πεθαίνουν.
-Όλα τα φυτά έχουν μέγεθος(size),αλλά τα Seedless φυτά έχουν πάντα μέγεθος 0.
-Στα φυτά που δεν αναφέρεται σε τί είδους πεδίο μπορούν να αναπτυχθούν ή να ευδοκιμήσουν, θεώρησα ότι μπορούν εκεί που μπορούν να φυτρώσουν.
-Στα Seeded φυτά, όταν φαγωθούν από ένα ζώο,η μείωση των καρπών, του φυλλώματος και της ζωής τους γίνεται κλιμακωτά με την σειρά που αναφέρεται στην εκφώνηση.
-Παρόλο που από τα σαρκοφάγα ζώα δεν χρησιμοποιείται και επειδή στην εκφώνηση αναφέρονται τιμές για αυτό το χαρακτηριστικό σε κάθε είδος σαρκοφάγου ζώου, όλα τα ζώα έχουν το χαρακτηριστικό neededFood το οποίο είναι
οι μονάδες τροφής που καταναλώνουν σε μία ημέρα. Στα σαρκοφάγα ζώα απλά αρχικοποιώ τις αντίστοιχες τιμές μέσω των constructors τους και απλά τις αυξάνω όταν αναπτύσσονται και δεν τις χρησιμοποιώ πουθενά αλλού. Ο μόνος
λόγος που το έκανα έτσι είναι για να υλοποιήσω όλες τις προδιαγραφές της εκφώνησης.
-Επειδή τα σαρκοφάγα ζώα χορταίνουν αμέσως αφού φάνε ένα άλλο ζώο, δεν χρειάζονται eatCount.Επομένως το eatCount σε όλα τα σαρκοφάγα ζώα είναι πάντα 1.

Όταν έκανα την εργασία πρώτα έφτιαξα τις κλάσεις για το οικοσύστημα, το MapGenerator και μετά για τα φυτά και τα ζώα.Έκανα αρχικά τις δοκιμές μου με μία πρόχειρη main και σταδιακά πρόσθεσα τις λειτουργίες σε όλες τις
κλάσεις και στο τέλος έφτιαξα τη main να διαβάζει τα δεδομένα με ελέγχους εγκυρότητας και να τρέχει την προσομοίωση με βάση αυτά.
